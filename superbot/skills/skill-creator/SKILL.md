# Skill Creator

## Creating Skills

Create a skill directory at `~/.superbot/workspace/skills/{skill-name}/` with:
- `SKILL.md` - Required skill definition

## SKILL.md Format

```yaml
---
name: skill-name
description: What this skill does
---
```

Keep skills concise. The agent is already smart.
